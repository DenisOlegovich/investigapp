# Spacings

Все измеряется `rems`, где `1rem = 10px`. Spacings в Tailwind CSS можно использовать например, как

`p-[n], px-[n], py-[n]` = padding

`m-[n], mx-[n], my-[n]` = margin

`space-y-[n]` or `space-x-[n]` = margin между элементами

`w-[n]` or `h-[n]` = width или height
